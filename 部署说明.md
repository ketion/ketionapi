========================================
部署步骤：
========================================

1. 上传文件到服务器:
   - new-api-modelscope-1.0.0.tar
   - docker-compose.yml

2. 在宝塔面板中操作:
   a. 安装 Docker 和 Docker Compose
   b. 进入终端，导入镜像:
      docker load -i new-api-modelscope-1.0.0.tar

3. 启动服务:
   docker-compose up -d

4. 访问地址:
   http://你的服务器IP:3000
